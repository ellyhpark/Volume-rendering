Volume Rendering
</h1>

A (work-in-progress) volume renderer that visualizes 3D Perlin noise.

- Bounding box geometry and *ShaderMaterial* provided by [three.js](https://threejs.org/)
- Use of the *Beer-Lambert* law to calculate sample point opacity and the front-to-back compositing equation to accumulate total color and opacity
- Layered and improved the look of noise by *Fractional Brownian Motion*
- User control for different volume densities

## Preview
Feel free to try out the [interactive demo](https://ellyhpark.github.io/Volume-rendering).
